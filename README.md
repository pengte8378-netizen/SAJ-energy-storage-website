# PRISM 工商业储能产品需求管理工具

面向产品总监和产品经理的内部需求台账与进度跟踪工具。

## MVP 功能

- 演示账号登录、会话保持与退出
- 仪表盘统计、负责人负荷、即将到期、延期与风险下钻
- 全部需求 / 我的需求、搜索、组合筛选、排序视图与分页
- 新建需求、状态流转、进展更新、计划日期调整原因校验
- 自动计算完成、延期、风险、阻塞与长期未更新
- 更新记录、关键字段变更历史和中文 CSV 导出
- 产品、预警阈值和团队时区基础设置
- 初始需求池为空，可由四位团队成员从真实需求开始录入

## 本地运行

要求 Node.js 22.13+ 与 pnpm。

```bash
pnpm install
pnpm dev
```

生产构建与测试：

```bash
pnpm build
pnpm test
```

默认地址：`http://localhost:3000/`
